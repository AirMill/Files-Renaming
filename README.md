# Files Renaming

A batch file renaming tool with preview-first workflow.

Built for photographers and users managing structured file systems.

---

## ✨ Features

- Batch rename across folders
- Preview before applying changes
- Subfolder support
- EXIF support (for images)
- Simple workflow
